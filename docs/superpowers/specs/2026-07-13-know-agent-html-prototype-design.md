# Know-Agent HTML 原型设计规格

## 目标

基于 `know-agent-ui` 当前已经实现的功能，生成一个与生产前端解耦的单页 HTML 原型，作为后续视觉与交互改版的设计底稿。

## 范围

- 登录态与退出登录。
- 智能助理：新建会话、历史会话、消息发送、工具调用记录、工具审批。
- 工作流：工作流卡片列表；PPT 生成作为其中一个工作流，进入后展示 Mermaid 执行流程和对话式生成状态。
- 知识库：搜索、状态筛选、文档表格、上传弹窗、文档详情、分块与向量化操作。
- 桌面端完整工作台布局，移动端提供可用的折叠导航与内容重排。

## 视觉方向

沿用当前前端的浅色 ultramodern 基调，以克制的中性色、青绿色主操作色和橙色状态强调构成层级。界面保持工具型产品的紧凑与可扫描性，不使用营销式大标题或装饰性卡片堆叠。

## 信息架构

左侧为品牌、一级导航、智能助理会话列表与用户入口；右侧为模块标题栏和工作区。工作流入口先展示卡片，PPT 详情复用智能助理式对话区。所有状态在同一 HTML 中以 mock 数据驱动，不请求后端接口。

## 交互约定

- 一级导航、会话切换、新增/删除、发送消息、工具审批均可点击；新增只创建草稿，首条消息提交后才进入历史。
- 工作流列表卡片可进入 PPT 详情；PPT 详情展示后端可返回的 Mermaid 图定义、渲染流程和对话式完成状态。
- 知识库支持搜索、筛选、打开详情、上传弹窗和状态推进。
- 退出后显示登录界面，登录可恢复工作台。
- 重要操作提供 toast、确认或明确的状态反馈。

## 验收

- 直接打开或通过静态服务器访问即可运行。
- 页面无控制台语法错误，主要操作可完成闭环。
- 1440x900 与 390x844 视口下无文字遮挡或核心控件溢出。
- 原型文件只写入 `know-agent-ui/原型/`，不修改生产前端。
